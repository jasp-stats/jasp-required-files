bf.bic {flexplot}R Documentation

Compute Bayes Factor from BICs

Description

Compute Bayes Factor from BICs

Usage

bf.bic(model1, model2, invert = F)

Arguments

model1

The first model to be compared

model2

The second model to be compared

invert

Should the BF be inverted?

Details

Given two models, the bf.bic function calculates the approximate bayes factor from the two models' BIC values

Value

The approximate Bayes Factor of the model comparison

Author(s)

Dustin Fife
